# Composables UI Sample app

A Compose Multiplatform sample app that showcases how to use the [Composables UI components](https://composables.com/ui)
combined with [Compose Unstyled](https://composables.com/composeunstyled)'s
in a real kind of app.

The app showcases:

- Phone/Tablet/Desktop design using the `Breakpoints` API.
- Design System token based theming using Compose Unstyled
- Light/Dark themes

## Notes

The app uses the device default fonts in order to feel as native as possible. However, as the web platform cannot
currently use system installed fonts, we use `Inter` instead, as it is a popular, modern looking font used in many web
apps.
